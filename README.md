# GnomeDB

GnomeDB is an in-memory DBMS (Database Management System) built on Node.js

## Installation

Clone the repository and run npm installation command

```
npm install @gnomedb/gnomedb
```

## Scripts

Run tests

```
npm run test
```

Generate typescript declarations

```
npm run types
```

Run linter

```
npm run lint
```
